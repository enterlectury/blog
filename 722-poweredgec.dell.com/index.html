PXE image

See also: PXE image. A network-bootable, provisioning PXE image is offered. This is a known, good image that all of the tools work in. The image contains simple, BASH and Perl script based methods of:

  • analyzing/inventorying the server
  • pulling down provisioning packages from an FTP server
  • pushing inventory and machine detection info back up
  • flash updating of BIOS/BMC/FCB, if desired/required
  • ... literally anything you can script!
